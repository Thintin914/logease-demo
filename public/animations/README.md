# Rive Animations

Place your `.riv` animation files in this directory.

## Example Structure:
- `realtime-monitoring.riv` - Animation for Real-Time Monitoring feature
- `log-search.riv` - Animation for Log Search feature
- `data-analytics.riv` - Animation for Data Analytics feature
- etc.

## How to use:
1. Export your animation from Rive editor as a `.riv` file
2. Place it in this `public/animations/` folder
3. Update the component to reference the correct file path
4. Update the state machine name if needed

## Tips:
- Keep file sizes reasonable for web performance
- Use state machines for interactive animations
- Test animations on different screen sizes
